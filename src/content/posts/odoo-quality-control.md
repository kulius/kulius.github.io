## 需求分析

製造業對品質管理有嚴格要求，需要從來料、製程到成品的全流程品質追蹤。本專案為電子零組件製造商開發完整的品質管理系統。

## 系統功能

### 1. 品質檢驗點設定 (quality_point)
- 多種檢驗類型支援
- 檢驗標準彈性設定
- 自動觸發檢驗任務

### 2. 多圖片檢驗記錄 (quality_point_test_type_multi_picture)
- 檢驗照片上傳與管理
- 缺陷圖片標註
- 歷史記錄追溯

### 3. 製程品管 (quality_mrp_workorder)
- 工作站品檢整合
- 即時異常通報
- SPC 統計製程管制

### 4. 檢驗報表
- 不良率統計分析
- 供應商品質評比
- 趨勢圖表視覺化

## 工作流程

```
來料 → IQC 檢驗 → 入庫
         ↓
       不合格 → 退貨/特採
         
製程 → IPQC 檢驗 → 續行
         ↓
       不合格 → 重工/報廢

成品 → FQC 檢驗 → 出貨
         ↓
       不合格 → 重檢
```

## 客戶效益

- 品質異常反應時間從 2 天縮短至 2 小時
- 不良品流出率降低 90%
- 客訴案件減少 60%

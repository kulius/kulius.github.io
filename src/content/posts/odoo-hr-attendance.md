## 專案需求

企業人資管理面臨多項挑戰：員工分散多據點、排班複雜、加班計算規則多變。本專案針對這些痛點開發完整的人資出勤解決方案。

## 核心模組

### 1. 出勤打卡整合 (sl_hr_attendance)
- 支援 ZK 生物辨識打卡機
- GPS 定位遠端打卡
- 異常出勤自動提醒

### 2. 排班管理 (sl_hrm)
- 視覺化班表編排
- 跨門市人力調度
- 個人行事曆整合

### 3. 加班與請假 (sl_hrm_overtime / sl_hrm_holiday)
- 符合台灣勞基法的加班費計算
- 多層級請假簽核流程
- 特休自動結算

### 4. 薪資計算 (sl_hrm_payroll)
- 自動化薪資計算
- 勞健保級距對照
- 年度所得申報整合

## 系統架構

```
┌─────────────────┐
│  打卡機/APP     │
└────────┬────────┘
         │
    ┌────▼────┐
    │ Odoo HR │
    ├─────────┤
    │ 出勤    │
    │ 排班    │
    │ 請假    │
    │ 薪資    │
    └─────────┘
```

## 導入成效

- 人資作業時間減少 70%
- 薪資計算錯誤率趨近於零
- 員工自助查詢提升滿意度
